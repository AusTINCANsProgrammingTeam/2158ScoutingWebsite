# 2158 Scouting Website

## Team 2158 ausTIN CAN's Scouting Website

> SASS (Bootstrap), Javascript (Node.js, Express.js)

### Installation

 - Install [Node.js](https://nodejs.org/en/download) (v24-25 compatible as of 12/20/2025)
 - Clone this repository
 - Navigate to repository root with a terminal
 - Run the following
    ```bash 
    npm install
    ```

### Usage

```bash
# Compile SCSS and run local server:
npm run dev
# Run only local server
npm run start
# Compile SCSS
npm run scss:build
```
